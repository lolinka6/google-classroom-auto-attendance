# Google Classroom Auto Attendance

Автоматическая отметка присутствия в Google Classroom для практикантов.

## Проблема

Каждый день до 10:30 нужно заходить в Google Classroom и отмечаться в опросе «Вы на практике? Живы-здоровы?» — поставить галочку «Да» и отправить. Если проспать или забыть — проблемы. Делать это вручную каждое утро надоело.

## Решение

Python-скрипт, который делает всё автоматически:

1. Открывает браузер (Chrome)
2. Логинится в Google Classroom под моим аккаунтом
3. Находит сегодняшний опрос по заголовку
4. Ставит галочку напротив «Да»
5. Отправляет ответ

Скрипт запускается по расписанию через планировщик Windows в 9:30 утра. Теперь я сплю спокойно 😴

## Технологии

- **Python 3** — язык программирования
- **Selenium** — автоматизация браузера
- **Task Scheduler (Windows)** — запуск по расписанию
- **Git** — контроль версий

## Как запустить у себя

1. Клонировать репозиторий
2. Установить Selenium: `pip install selenium`
3. Скачать ChromeDriver под вашу версию Chrome
4. Добавить логин и пароль в переменные окружения (не коммитить!)
5. Запустить скрипт: `python main.py`
6. Настроить планировщик Windows для ежедневного запуска
